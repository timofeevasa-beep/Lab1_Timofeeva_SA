# Lab1_Timofeeva_SA
# Лабораторная работа №1

## Описание проекта
Программа для определения типа треугольника по длинам трех сторон и вычисления координат его вершин для отрисовки в поле 100x100 пикселей.

## Функциональность
- Проверка валидности входных данных (вещественные положительные числа)
- Определение типа треугольника: равносторонний, равнобедренный, разносторонний
- Вычисление координат вершин с масштабированием под 100x100
- Логирование успешных и неуспешных запросов в файл и консоль

## Технологии
- Язык: Python 3.x
- Логирование: стандартный модуль logging

## Запуск программы
python main.py
